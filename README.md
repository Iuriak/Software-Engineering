用例图包含的元素

1. 参与者(Actor)——与应用程序或系统进行交互的用户、组织或外部系统。用

一个小人表示。

2. 用例(Use Case)——用例就是外部可见的系统功能，对系统提供的服务进行描

述。用椭圆表示。

3. 子系统(Subsystem)——用来展示系统的一部分功能，这部分功能联系紧密。
